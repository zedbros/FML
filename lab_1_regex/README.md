# Lab 1 — Expressions régulières

Laboratoire d'introduction aux expressions régulières en Python, basé sur le module `re`.

Pour ce labo, vous avez droit au LLM level **L1: créer du matériel pédagogique**.

## Code

- `regex_lab.py` — Fichier principal à compléter (classe `RegexExercises`)
- `regex_test.py` — Tests unitaires et classe `Helper` pour le débogage
- `searcher.py` — Utilitaire de recherche dans les fichiers texte


## Fichiers

- `BleakHouse.txt` — Texte de *Bleak House* (Dickens), utilisé comme corpus
- `numbers.txt` — Fichier de nombres pour les exercices sur les formats numériques

## Utilisation

Complétez les méthodes de `regex_lab.py` puis lancez les tests :

```bash
python regex_test.py
```

Astuce: dans le fichier `regex_test.py`, vous pouvez décommenter la ligne `Helper.main()` pour visualiser les résultats (matches) de vos expressions régulières.
